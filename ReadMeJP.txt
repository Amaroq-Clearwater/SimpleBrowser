# SimpleBrowser by 片山博文MZ

SimpleBrowserは、Windows用のシンプルなウェブブラウザです。

Webページを閲覧できます。

対応環境: Windows XP/Vista/7/10

## ショートカット キー

| キーの組み合わせ | コマンド                   |
|------------------|----------------------------|
| Ctrl+R           | 再読み込み                 |
| F5               | 再読み込み                 |
| Alt+Left         | １つ戻る                   |
| Alt+Right        | １つ進む                   |
| Alt+Home         | ホームへ移動               |
| Esc              | 停止                       |
| Alt+D            | アドレスバーへ移動         |
| Ctrl+P           | 印刷...                    |
| Shift+Ctrl+P     | 印刷!                      |
| Ctrl+U           | ソースを見る               |
| Alt+F            | 点々メニューを表示         |
| Ctrl+N           | 新しいウィンドウ           |
| Ctrl+K           | キオスクモードを切り替える |

キオスクモードでは、点々メニュー ("...") と印刷プレビューは
セキュリティ上の理由から利用できません。
